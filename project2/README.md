# 🔐 Basic Encryption & Decryption using Caesar Cipher

## 📌 Project Overview

This project is a Python-based implementation of the **Caesar Cipher**, one of the simplest and most well-known encryption techniques in cryptography. It allows users to encrypt and decrypt text using a custom shift key while preserving spaces, numbers, and special characters.

The project was developed as part of my Cyber Security training to understand the fundamentals of data confidentiality and basic encryption algorithms.

---

## ✨ Features

- Encrypt any text using the Caesar Cipher algorithm
- Decrypt encrypted text back to its original form
- User-defined shift key
- Supports both uppercase and lowercase letters
- Preserves spaces, numbers, and special characters
- Simple and user-friendly command-line interface

## 🔒 How Caesar Cipher Works

The Caesar Cipher encrypts text by shifting each alphabetic character by a fixed number of positions in the alphabet.

Example:

```
Original : HELLO

Shift Key : 3

Encrypted : KHOOR
```

During decryption, the same shift is applied in the opposite direction to recover the original text.
